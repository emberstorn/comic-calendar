# Comic Release Calendar

A shared calendar of upcoming comic book releases, updated regularly.

### 👉 [View the calendar](https://emberstorn.github.io/comic-calendar/)

https://emberstorn.github.io/comic-calendar/

---

## What's here

A monthly calendar of new comic releases pulled together from several
publishers. Wednesdays — new comic day — are highlighted, and every release
links through to its details.

You can:

- **Browse by month** — use the ‹ › arrows to move between months
- **Switch views** — month grid or a simple list by date
- **Filter by publisher** — tap a publisher to show or hide its books
- **Filter by type** — comics vs. graphic novels / collected editions
- **Search** — by title, series, or creator
- **Tap any release** — to see the cover, description, creators, and price

Nothing to install — it's just a web page. Open the link on your phone or
computer and you're set. Bookmark it and it'll always show the latest data.

## Where the data comes from

Releases are gathered from publisher sites and a comics retailer, then merged
so each issue appears once with the best available details. Dates follow the
publishers' announced on-sale dates and can shift as plans change, so treat
this as a helpful guide rather than a guarantee.

## A note for the curious

This site is built by a small set of Python scripts that scrape release info,
combine it, and generate the calendar page you see. The page is plain HTML —
no accounts, no tracking, just the calendar.
